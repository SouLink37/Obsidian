### 2. public，external，internal，private

| 修饰符      | 内部调用 | 外部调用 | 继承调用 | 用途    |
| -------- | ---- | ---- | ---- | ----- |
| public   | ✅ 可以 | ✅ 可以 | ✅ 可以 | 完全公开  |
| external | ❌ 不行 | ✅ 可以 | ❌ 不行 | 只给外部用 |
| internal | ✅ 可以 | ❌ 不行 | ✅ 可以 | 内部使用  |
| private  | ✅ 可以 | ❌ 不行 | ❌ 不行 | 严格私密  |

Interface 就是 Solidity 中的"虚函数"概念

### 3. 类比对比：

| 特性     | Interface | 虚函数      |
| ------ | --------- | -------- |
| 定义规范   | ✅ 只定义函数签名 | ✅ 定义虚函数  |
| 无具体实现  | ✅ 没有代码    | ✅ 没有代码   |
| 强制实现   | ✅ 继承者必须实现 | ✅ 子类必须重写 |
| 通过多态调用 | ✅ 可以      | ✅ 可以     |
|        |           |          |
